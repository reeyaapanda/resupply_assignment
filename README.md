# Resupply Assignment  
Evaluation of the Item Database Feature  

## Overview  
This project evaluates Resupply’s Item Database feature by comparing donor-selected items with driver-recorded items. The analysis examines recategorization patterns, recycling fees, and size-based item performance.

## Files  

- `dls_final.sql`  
  Creates the cleaned donation-level dataset and recategorization indicators.

- `ilp_final.sql`  
  Builds the item-level performance view with recategorization and recycling rates.

- `resupply_assignment.Rmd`  
  Code for the full analytical report including methodology, modeling, and recommendations.

- `final_report.html`  
  Rendered HTML version of the final report.

## Methods  
- SQL for data extraction and aggregation  
- R for analysis and modeling  
- Logistic regression to study associations between recategorization, size, and recycling fees  

Note: This is an observational analysis. Results reflect associations, not causal effects.
